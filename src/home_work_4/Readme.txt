DataContainer решает следующие задания:
№ 4. Метод add добавляет данные во внутреннее поле data и возвращает номер позиции в которую были вставлены данные
№ 5. Метод get возвращает из поля data переданый объект на предыдущем шаге через метод add
№ 6. Метод getItems возвращает поле data
№ 7. Метод indexDelete который удаляет элемент из поля data по индексу
№ 8. Метод itemDelete который удаляет элемент из поля data.
№ 9. Метод sort сортировки данных записанных в поле data
